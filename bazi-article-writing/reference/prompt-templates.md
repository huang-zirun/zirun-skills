# 提示词模板

用户不通过 Skill 而直接与模型对话时，可使用下列提示词以对齐风格与结构。仅在需要复制完整模板时查阅。

## 目录

1. [通用模板](#通用模板)

## 通用模板

```
请写一篇八字/命理类文章，主题是：[主题，如：什么日主最需要性和谐 / 偏印的底层心性]。

要求：
1. 开头：设问+一句答案（如「什么日主最需要XX？我认为是XX人」）或总括+核心问题，一句话定调。
2. 理论依据：从五行/十神原理一步步推，写出十神关系（克谁、被谁制、与谁配），不只下结论。
3. 条件细分：落到透干、坐支、神煞等可对应命局的具体条件（如水火相见、食伤旺、坐沐浴），不泛泛而谈。
4. 对比：与相关十神/日主对比时，明确与谁比、克谁被谁制。
5. 举例：至少一处类型化代表或生活化场景，不编具体八字或古籍。
6. 语气与节奏：真的、尤其是、注意了、因此、所以等高频出现；每段 1～3 句，短段；可适度口语梗。
7. 文末建议：具体、可操作、一句能记住，避免只写「理性规划」等抽象收束。
8. 篇幅：严格 **600–900 字**（按中文字符计）。
9. 格式：**纯正文**，不要使用任何 Markdown（无标题、加粗、列表、代码、链接等），仅用换行分段；主题适合分块时可加少量小标题。
```

## 带范文对齐的提示词

当用户已有一篇或多篇范文，希望严格对齐其风格与结构时使用：

```
请按下面文章的风格和结构，写一篇命理类文章。

主题：[你的主题]

风格要求（参考范文）：
- 开头设问+一句答案或核心问题；理论一步步推、条件细分到透干/坐支/神煞；对比写十神关系；至少一处举例；语气词密、短段；文末具体可操作建议。
- 术语准确（十干、十神、地支、神煞等），语气可口语化（真的、容易、尤其是）。
- 不编造古籍或名人八字。
- 篇幅 600–900 字；输出为纯正文，不要使用任何 Markdown 格式。

[此处粘贴范文全文]
```

## 严格复刻范文风格（一模一样的写法）

当需要 LLM 写出与范文**高度一致**的结构、语气和节奏时，使用下面的完整约束：

```
请写一篇八字/命理类文章，主题是：[主题]。

必须做到：

【开头】
- 第一段就要亮出观点或设问。例如：设问+直接给答案（如「什么日主最需要XX？我认为是XX人」），或先总括再抛核心问题（如「XX是十神中最……的，但它的XX之性始终贴身相伴，这该如何理解呢？」）。

【结构】
- 按顺序组织：理论依据（从五行/十神一步步推，写出克谁被谁制）→ 条件细分（透干、坐支、神煞等可对应命局的具体条件）→ 对比其他十神或日主（明确与谁比、在十神中的位置）→ 至少一处举例（类型代表或生活场景，不编具体八字）→ 文末总结或给读者建议（具体、可操作、一句能记住）。

【语气与用词】
- 中文简体，介于科普与网文之间。多用「真的」「容易」「尤其是」「注意了」「因此」「所以」做强调和收束。
- 可用「我认为」「一般来讲」，且后面必须跟命理逻辑支撑。
- 避免绝对化，用「往往」「容易」「多数」留余地。可适度用口语梗（如「床头不和床尾和」）增加网文感。

【段落】
- 每段 1～3 句成段，短段制造网文节奏。在关键结论前用「因此」「所以」「注意了」做小结再展开。

【术语与禁忌】
- 十天干、十神、五行、地支、神煞等术语准确、体系统一。不编造古籍原文，不虚构名人八字；可举某类格局的代表（如某历史人物仅作类型说明，不写具体命盘）。

【篇幅与格式】
- 篇幅：[可选：600–900 字 / 与范文相近时可按范文长度]。
- 格式：纯正文，仅换行分段。若主题适合分块可加少量小标题（如「XX的悟性为何如此透彻？」），否则不用任何 Markdown（无 #、**、列表、链接等）。
- 文末建议须具体、可操作、一句能记住，避免只写「理性规划、设上限」等抽象收束。
```

若希望**完全仿写**某一篇，把该范文全文贴在提示词最后，并补一句：「请严格模仿上面这篇的结构、段落长短、语气词密度和收束方式来写新主题。」
